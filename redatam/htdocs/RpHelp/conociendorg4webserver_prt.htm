Conhecendo R+G4 Web Server
[Main] [Next]

   A quarta geração de REDATAM (RG4) incorpora uma nova funcionalidade para o desenvolvimento de aplicações orientadas a facilitar o acesso dos usuários à informação relevante para o levantamento de estudos e realização de diagnósticos cuja temática dependerá do conteúdo das bases de dados disponíveis. Tal funcionalidade permite armazenar a informação num servidor de páginas web que pode receber solicitações de processamento para bases de dados em formato REDATAM, executar as instruções necessárias e devolver o resultado requerido.

   Desde a versão 1.2, o REDATAM possue a capacidade de acessar bases de dados localizadas num servidor, de tal maneira que os usuários que trabalham numa instituição determinada somente necessitam instalar o software e executar os processamentos pertinentes para a obtenção dos resultados que se requeiram. Ainda que este procedimento ajuda a liberar espaço nas máquinas locais e a organizar as bases existentes em um único lugar, ele apresenta um custo associado, definido pelos recursos que devem ser atribuídos tanto para a máquina local como ao próprio servidor para resolver cada solicitação de processamento que cada usuário solicite.

   A nova funcionalidade de REDATAM usa a tecnologia e linguagem desenvolvidas no contexto da Intranet e Internet por meio de servidores de páginas web. A diferença com o descrito no parágrafo anterior é que neste caso a aplicação e o processamento dos dados são executados pelo servidor, o que significa que os usuários não necessitam ter na sua máquina local nem as bases de dados que desejam consultar nem ter instalado REDATAM para obter os indicadores ou tabelas que queiram em um momento determinado.
